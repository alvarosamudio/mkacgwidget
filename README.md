# LinuserDesktop
该项目是为了练习QT及QML代码。
计划实现的功能有：
    显示一句动漫台词
    电台播放
    伪小娜
    伪春菜
    天气播报

脑洞：
    开机提醒和关机提醒，不定时通知用户是否要休息。
    调戏程序
